# 🕊️ Православные храмы Владивостока

Информационный портал о православных храмах Владивостока с возможностью просмотра расписания, отзывов и взаимодействия с администрацией.

## 🚀 Функционал

- 📋 Просмотр списка всех храмов города
- 🔍 Детальная страница каждого храма с историей, расписанием и картой
- 💬 Система отзывов с рейтингом и модерацией
- 📅 Управление расписанием богослужений
- 📧 Форма обратной связи через EmailJS
- 👑 Админ-панель для управления контентом

## 🛠️ Технологии

### Backend
- Node.js + Express
- SQLite3
- JWT для авторизации

### Frontend
- React 18
- React Router v6
- Axios
- Framer Motion (анимации)
- React Icons
- EmailJS
